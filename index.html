FIFA Flag Icons CSS

Contains a collection of flags for all the football national teams recognized by FIFA. The flags are indexed using FIFA codes instead of ISO 2 codes. It was created to support my UFWC fan website.

Forked from the awesome project flag-icon-css. Having such amazing collection of flags made all easier. When I have the time to submit a pull request to this project to implement the ability to provide custom classes to existing flags maybe this project will cease to exist
